Google Sheet Live Sync

One-time setup: install the sync script on the shop Google Sheet (instructions come with the app), then paste the Web App URL it gives you here. After that, every log, date, and note change is pushed to the sheet automatically over wifi — it fills the APP LOG, APP ALL DATES and APP PARTS USED tabs.

Excel Export

Makes an .xlsx file with one sheet per kart in the same layout as the shop spreadsheet (DATE, ACTION COMPLETED, parts used, MECHANIC, NOTES + tracked dates), plus an ALL DATES summary sheet.

Backup & Restore

Backup copies all log data as text. Save it somewhere safe (email it to yourself). Restore pastes it back in — use this to move data to another device. NOTE: photos are too big for the text backup — they stay on this device only.
